## 📌 1️⃣ `morgan` – HTTP Logger

`morgan('dev')` logs requests in terminal.

When you hit:

```
GET /users
```

Terminal shows something like:

```
GET /users 200 5.123 ms - 45
```

### 🔥 Practice Task 1

Create a simple Express server:

```js
const express = require('express');
const morgan = require('morgan');

const app = express();

app.use(morgan('dev'));

app.get('/', (req, res) => {
  res.send('Hello Anudeep');
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
```

👉 Open browser → `http://localhost:3000`
👉 See terminal logs

---

## 📌 2️⃣ `helmet` – Security Middleware

`helmet()` adds security headers.

It protects against:

* XSS attacks
* Clickjacking
* MIME sniffing
* Some basic vulnerabilities

### 🔥 Practice Task 2

After starting server:

Open browser →
Right click → Inspect → Network → Click request → Headers

You will see headers like:

* `X-Content-Type-Options`
* `X-Frame-Options`

That is helmet working.

## 🔥 Practice Challenge 1 – Custom Middleware

Create your own logger:

```js
app.use((req, res, next) => {
  console.log("Time:", new Date().toISOString());
  console.log("Method:", req.method);
  console.log("URL:", req.url);
  next();
});
```

Understand:

* `req`
* `res`
* `next()`

Very important concept.

---

## 🔥 Practice Challenge 2 – Protect a Route

Create middleware:

```js
function checkAuth(req, res, next) {
  const token = req.query.token;

  if (token === "1234") {
    next();
  } else {
    res.status(403).send("Unauthorized");
  }
}

app.get('/admin', checkAuth, (req, res) => {
  res.send("Welcome Admin");
});
```

Test:

```
http://localhost:3000/admin?token=1234
```
